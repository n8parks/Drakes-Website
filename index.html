What Is Plugdata?

Plugdata is a free, visual Pure Data environment for creating and exporting audio patches as plugins (VST3, AU, LV2, CLAP).

Installing Plugdata

  1. Go to plugdata.org and download the installer for your OS (Windows, macOS, or Linux).
  2. Run the installer and launch Plugdata.

Step-by-Step Plugin Tutorial

Creating “GainControlPlugin”

  1. Open Plugdata and make a new patch named gain_control.pd.
  2. Add adc~ 1 2, *~, gain parameter, and dac~ 1 2.
  3. Test with ▶. Adjust gain knob.
  4. File ▶ Compile… ▶ DPF Audio Plugin ▶ VST3 ▶ Compile. Save the resulting .vst3.
  5. Load in your DAW and play with the new plugin!
